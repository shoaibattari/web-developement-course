Chapter 7: JavaScript Arrays

This chapter explains how to create, access, and manipulate arrays in JavaScript, along with various array methods including the rest operator.

Array Methods

  • Push() - Adds an element to the end of the array.
  • Pop() - Removes the last element and returns it.
  • ToString() - Converts the array to a string.
  • Concat() - Joins multiple arrays and returns the result.
  • Unshift() - Adds an element to the beginning of the array.
  • Shift() - Removes the first element and returns it.
  • Slice() - Returns a piece of the array from a start index to an end index.
  • Splice() - Changes the original array by adding, removing, or replacing elements.
  • Rest Operator ( ... ) - Used to collect multiple elements into an array or spread array elements.
