JavaScript Variables

Variables are containers for data in memory. Rules for naming variables:

  • Case-sensitive.
  • Allow only letters, digits, underscores, and $.
  • Start with a letter, underscore, or $.
  • Avoid reserved words.

Variable Declaration Types

var: Can be re-declared and updated; global scope.

let: Cannot be re-declared but can be updated; block scope.

const: Cannot be re-declared or updated; block scope.

Examples

        
            // Using var
            var name = "John";
            console.log(name); // Output: John
            name = "Doe";
            console.log(name); // Output: Doe

            // Using let
            let age = 25;
            console.log(age); // Output: 25
            age = 30;
            console.log(age); // Output: 30

            // Using const
            const country = "USA";
            console.log(country); // Output: USA
            // country = "Canada"; // This will throw an error
